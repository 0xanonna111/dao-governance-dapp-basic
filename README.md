# DAO Governance DApp Basic

This project helps beginners learn how decentralized governance works using blockchain.

You will learn:
- What a DAO is
- How proposals are created
- How voting works on-chain
- How to connect a wallet
- How to read voting results

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/dao-governance-dapp-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- dao.js = DAO contract functions
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Create proposal
- Vote on proposal
- Show results
- Simple DAO flow

## Learning Goal
This project explains how DAOs manage decisions in a decentralized way.
